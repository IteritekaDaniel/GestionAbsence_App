# AbsencesGestion

Application desktop développée en Python pour la gestion des absences scolaires.  
Elle permet de suivre les étudiants, enregistrer les absences, notifier les parents et générer des rapports.

---

## Fonctionnalités

- Gestion des étudiants (CRUD)
- Suivi des absences (justifiées / non justifiées)
- Gestion des parents
- Notifications email
- Système de validation des absences
- Alertes configurables
- Multi-administrateurs avec rôles
- Authentification sécurisée avec 2FA
- Rapports PDF et CSV
- Journal des actions (audit)

---

## Technologies

- Python  
- CustomTkinter  
- SQLite  
- Matplotlib  
- ReportLab
