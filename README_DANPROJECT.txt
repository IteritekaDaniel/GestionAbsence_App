⌨️ RACCOURCIS CLAVIER ESSENTIELS
═══════════════════════════════════════════════════════════════════════════════

Navigation:
  Ctrl + 1        → Étudiants
  Ctrl + 2        → Absences
  Ctrl + 3        → Parents
  Ctrl + 4        → Demandes
  Ctrl + 5        → Statistiques
  Ctrl + 6        → Rapports
  Ctrl + 7        → Alertes
  Ctrl + 8        → Audit

Actions:
  Ctrl + Shift + T  → Basculer Thème (Clair/Sombre)
  Ctrl + Shift + S  → Paramètres
  Ctrl + Shift + Q  → Déconnexion

💡 CONSEILS D'UTILISATION
═══════════════════════════════════════════════════════════════════════════════

1. Utilisez les raccourcis clavier pour aller plus vite
2. Activez le 2FA dans les Paramètres pour la sécurité
3. Exportez régulièrement vos données
4. Consultez les graphiques pour analyser les tendances
5. Changez de thème selon votre préférence (risque clair/obscur)


🎯 PROBLÈMES COURANTS
═══════════════════════════════════════════════════════════════════════════════

❌ "ModuleNotFoundError: No module named 'customtkinter'"
✅ Solution: pip install -r requirements.txt

❌ "L'application plante au démarrage"
✅ Solution: Redémarrez: python main.py

❌ "Les boutons n'ont pas les bonnes couleurs"
✅ Solution: Assurez-vous que theme.py est correctement installé
